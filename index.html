WEB

구글 웹 툴킷(Google Web Toolkit, GWT)은 웹 개발자가 복잡한, 자바스크립트 프론트엔드 애플리케이션을 자바로 만들고 관리할 수 있게 하는 오픈 소스 도구 집합이다. 몇 가지 네이티브 라이브러리를 포함하여, 모든 것이 자바 소스로 이루어져 있으며, 이 소스는 GWT 앤트 빌드 파일이 포함하고 있는 지원 운영 체제 어느 곳에서나 빌드가 가능하다. 아파치 라이선스 버전 2.0으로 허가되어 있다.[1] GWT는 재사용성, 공통 웹 개발 작업으로의 효율적인 접근 (비동기 원격 프로시저 호출), 역사 관리, 북마크, UI 축약, 국제화, 크로스 브라우저 이식을 강조한다.
